# mxgraph

记录 api

## graph

- graph.zoomIn() // 放大
- graph.zoomOut() // 缩小
- this.graph.view.scale // 当前缩放比例

## mxEvent

- mxEvent.consume(evt) // 消耗给定的事件
  - evt 要消耗的本机事件。
  - preventDefault 可选的 boolean，以防止默认事件。默认为 true。
  - 停止传播 布尔值选项，用于停止事件传播。默认为 true。
